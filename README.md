Example

## REQUIRED INSTALLATION

### Flask

	pip install flask

## SETUP

Set `FLASK_APP` to `app`<br>
Set `FLASK_ENV` to `development`

	$env:FLASK_APP = "app"
	$env:FLASK_ENV = "development"

## HOW TO RUN

While in directory with `app.py`

	flask run

Access the site on your machine's local host at `localhost:5000`
